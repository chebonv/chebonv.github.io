# Vincent Chebon - Portfolio Website

A modern, responsive portfolio website showcasing my professional experience, skills, and projects as a Full-Stack Developer.

## 🚀 Features

- Responsive design that works on all devices
- Modern and clean UI with smooth animations
- Interactive project showcase with filtering capabilities
- Skills section with categorized technical expertise
- Detailed experience timeline
- Contact form for direct communication
- Social media integration

## 🛠️ Tech Stack

### Frontend
- HTML5
- CSS3 (with Tailwind and Bootstrap)
- JavaScript
- Font Awesome Icons
- Google Fonts (Montserrat and Open Sans)

### Backend
- Laravel
- FastAPI
- Various databases (Oracle, PostgreSQL, MySQL)

### Mobile Development
- Flutter
- Java (Android)

### Tools & Others
- Git
- Docker
- Linux
- Cloud Platforms (AWS, Azure, GCP, Digital Ocean)
- AI/ML & LLM

## 📁 Project Structure

```
my-portfolio-website/
├── index.html          # Main HTML file
├── css/
│   └── style.css       # Custom styles
├── js/

├── images/             # Project images
```

## 🎯 Key Sections

1. **Hero Section**
   - Professional introduction
   - Call-to-action buttons
   - Social media links

2. **About Section**
   - Professional summary
   - Downloadable CV

3. **Projects Section**
   - Filterable project showcase
   - Categorized by type (Web Apps, Mobile Apps, CMS Website)

4. **Skills Section**
   - Frontend technologies
   - Backend technologies
   - Tools & other expertise

5. **Experience Section**
   - Detailed work history
   - Education background

6. **Contact Section**
   - Contact form
   - Direct contact information
   - Social media links

## 🔧 Setup & Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/chebonv/my-portfolio-website.git
   ```

2. Open `index.html` in your preferred web browser

## 📝 Customization

- Update personal information in `index.html`
- Modify styles in `css/style.css`
- Add or update projects in the projects section
- Update skills and experience as needed
